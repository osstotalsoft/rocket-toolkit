# rocket-toolkit 👩‍🔧
This is a collection of plugins and other GraphQL utilities.

## Packages
  - [`apollo-logger`](./packages/zion#readme)

## Build
```javascript
yarn install
yarn build
```

## Test
```javascript
yarn test
```
